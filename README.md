# Stora

> 🚧 **Status: In Progress** — This project is currently under active development.

Stora is a multi-tenant ecommerce platform where users can easily create tenants (companies), link themselves as owners, invite staff members, manage customers, products, and combos — all from a single platform. Users can also serve as staff or customers for other tenants.

## Planned Features

- **Multi-Tenancy** — Create and manage multiple tenants/companies, each with its own settings, branding, and subscription plan.
- **Membership & Roles** — Assign roles such as owner, co-owner, admin, or staff within each tenant.
- **Staff Invitations** — Invite users to join a tenant via email with role-based access.
- **Product & Combo Management** — Create products, manage stock, and bundle products into combos.
- **Orders & Appointments** — Record customer orders and schedule appointments with staff.
- **Subscription Plans** — Support for Basic, Pro, and Enterprise tiers with configurable feature limits.
